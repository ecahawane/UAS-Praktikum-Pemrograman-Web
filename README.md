# UAS Praktikum Pemrograman Web

## Sistem Peminjaman Infokus Berbasis Web Menggunakan Laravel

Project UAS mata kuliah Praktikum Pemrograman Web menggunakan framework Laravel.

### Fitur
- Authentication Login & Register
- CRUD Data Infokus
- CRUD Peminjaman
- Dashboard
- Middleware Auth
- Role Admin dan User

### Teknologi
- Laravel
- MySQL
- Bootstrap
- PHP
